Pseudo-classes

Uma Pseudo-classes pseudo-classes é uma palavra-chave adicionada ás declarações de um seletor após um sinal de doios pontos e especificam um estado especial de um elemento.

Existem várias pseudo-calsse para estilos podemos citar:

  • : hover,

    • corresponde quando o usuário designa um elemento com um dispositivo apontador, mas não necessariamente o ativa

  • : visited,

    • O :visited é uma Pseudo-classe do CSS para indicar se o link em questão já foi visitado pelo usuário. Por questões de privacidade, os styles que podem ser modificados usando este seletor são muito limitados.

  • :active,

    • A pseudo classe : active é usada para adicionar um efeito especial a um elemento ativado. Ao definir pseudo-classes em um bloco << STYLE >>, os seguintes pontos devem ser tomados cuidado - a: hover DEVE vir depois de um: link e a: visitado na definição de CSS para ser eficaz.

  • :checked,

    • Checked Este é utilizado para definir um estilo para quando um checkbox está selecionado. Também pode ser utilizado na tag << option >> do elemento << select >>, para estilizar a opção selecionada

  • :empty,

    • :empty é utilizado para selecionar os elementos vazios, elementos sem qualquer conteúdo ou elementos que contenham apenas comentários. É útil para ocultar elementos vazios, por exemplo, que possam causar espaçamentos indesejados.

  • :focus,

    • focus foi introduzida nas CSS 2 e destina-se a aplicar estilos ao elemento HTML que receba o foco. A sintaxe da regra de estilo é: E:focus {propriedade:valor;} Onde E é um elemento HTML qualquer. O emprego mais comum desta pseudo-classe é em campos do tipo input para textos e campos do tipo textarea, em formulários.
